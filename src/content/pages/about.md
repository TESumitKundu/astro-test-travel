# About TrailNest Tours

TrailNest Tours is a static travel website built around seasonal tour packages, clear itineraries, and Git-managed content.

Editors can add tours, upload images, update day-wise descriptions, and adjust pricing through Pages CMS. Every change is saved as Markdown or JSON in GitHub.

This approach keeps the travel site fast and simple while still allowing content updates without WordPress, a database, or a server-side CMS.
